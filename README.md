# Harmandeep · Portfolio

Personal portfolio website. Live at [harmandeep2993.github.io](https://harmandeep2993.github.io)

Built with plain HTML, CSS, and JavaScript. No frameworks.

## Features

- Responsive layout for mobile and desktop
- DM Serif Display + DM Sans fonts
- Scroll reveal animations
- Dante AI chatbot to answer questions about my experience
- Hire Me button with pre-filled email
- Resume available on request

## To Change Colors or Fonts

Open `assets/css/variables.css` and update the values at the top of the file. All preset color schemes are included as comments.
